Project Images and Description

Trade History Parsing and Flattening

This block of code defines a function safe_eval() to safely evaluate strings in the 'Trade_History' column of the dataset using ast.literal_eval(). It converts valid string representations of Python data structures (like lists or dictionaries) into actual Python objects. Invalid entries are converted to None and dropped from the DataFrame. The function extract_trade_data() flattens each trade entry in the 'Trade_History' column, adding the associated 'Port_IDs' to each trade and creating a list of all individual trades. The flattened trades are then combined into a new DataFrame 'df_trades' containing all trades with relevant details, making the data easier to analyze and process.

Weighted Scoring System

This block defines a dictionary weights, which assigns different importance 'weights' to various trading performance metrics such as 'Total_ROI', 'Total_PnL', 'Sharpe_Ratio', 'Win_Rate', and 'Max_Drawdown'. The weights are used later in the code to calculate a composite rank score for each account. The goal is to evaluate accounts based on a combination of performance factors, with different aspects contributing differently to the final rank.

Metrics DataFrame Creation

This block constructs a new DataFrame metrics that aggregates performance metrics for each trading account. It computes metrics like 'Total_PnL', 'Total_ROI', 'Sharpe_Ratio', 'Max_Drawdown', 'Win_Rate', and the count of total and winning positions for each account (grouped by 'Port_IDs'). The metrics are rounded for readability, and the DataFrame is reset to remove any previous index. This resulting DataFrame is a structured summary of key performance indicators for each account, providing the foundation for further analysis or ranking.
